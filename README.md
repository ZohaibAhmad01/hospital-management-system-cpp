# Hospital Management System (C++)

A console-based hospital management application developed in C++ using object-oriented programming principles and file handling.

## Features

* Add patient records
* View all patients
* Search patient by ID
* Delete patient
* Book appointments

## Concepts Used

* Object-Oriented Programming
* Inheritance
* File Handling
* Modular Code Structure

## Project Structure

* Patient class for patient records
* Hospital class for patient management
* Appointment class for booking system
* Doctor class for inheritance demonstration

## Files

* patients.txt → stores patient records
* appointments.txt → stores appointment records

## Compile

g++ main.cpp Patient.cpp Hospital.cpp Appointment.cpp -o hospital

## Run

./hospital
